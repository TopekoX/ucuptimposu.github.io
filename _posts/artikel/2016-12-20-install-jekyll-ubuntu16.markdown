Step 1. Udate & upgrade system lewat terminal.

```
sudo apt-get update
sudo apt-get upgrade
```

Step 2. Install Ruby.

```
sudo apt-get install ruby-full
```

Step 3.Install Jekyll.

Install Jekyll menggunakan `gem`:

```
gem install jekyll
```

Cek versi jekyll

```
jekyll -v
```
